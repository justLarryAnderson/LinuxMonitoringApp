System Monitor - это графическое десктопное приложение для операционных систем Linux, разработанное на C++ с использованием библиотеки GTKmm.
Приложение предоставляет удобный интерфейс для мониторинга системных процессов и выполнения различных команд Linux.
Оно сочетает в себе функциональность системного монитора и терминала, позволяя пользователям в реальном времени отслеживать состояние системы и управлять ею через графический интерфейс.

Основной функционал

    1. Мониторинг процессов: Отображение списка запущенных процессов с информацией о PID, имени, использовании CPU, памяти и статусе

    2. Поиск процессов: Фильтрация процессов по имени в реальном времени

    3. Выполнение команд: Возможность выполнения любых Linux-команд с поддержкой конвейеров и сложных выражений

    4. Асинхронное выполнение: Команды выполняются без блокировки пользовательского интерфейса

    5. Визуальная обратная связь: Цветная индикация статуса выполнения команд (зеленый - успех, красный - ошибка)

    6. Локализованный интерфейс: Полная поддержка русского языка

    7. Темная тема: Стильное оформление окна справки

Технические требования:

    Операционная система: Linux (Ubuntu, Debian, Fedora, etc.)

    Зависимости:

        1. g++ (версия 7 или выше)

        2. GTKmm 3.0

        3. pkg-config

        4. make


1. Установка зависимостей (Ubuntu/Debian):

sudo apt-get update
sudo apt-get install build-essential libgtkmm-3.0-dev pkg-config

Для Fedora/RHEL:
sudo dnf install gcc-c++ gtkmm3.0-devel pkg-config make

Сборка и запуск

1. cd SystemMonitor

2. make

3. ./system-monitor

Пересборка после изменений

# Очистка предыдущей сборки и полная пересборка
make clean
make

# Или просто пересборка (если не было изменений заголовочных файлов)
make

./system-monitor
